# dotfiles

## Setup

```sh
stow --no-folding htop -t ~ -v
stow --no-folding stow -t ~ -v

stow --no-folding termite -t ~ -v
stow --no-folding alacritty -t ~ -v

stow --no-folding tig -t ~ -v
stow --no-folding tmux -t ~ -v
```
